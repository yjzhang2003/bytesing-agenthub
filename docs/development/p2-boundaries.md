# P2 Boundaries

These capabilities are explicitly outside MVP and should not be implemented as part of the control-plane foundation.

## GitHub Integration

P2 may add GitHub OAuth, repository linking, branch creation, commits, push, pull request creation, PR review comments, and CI status.

## Cloud Runtime

P2 may add cloud-hosted runtime devices that can clone repositories and execute agent tasks remotely. MVP only models `runtimeKind = cloud` as a future-compatible field.

## Deployment

P2 may add static site deployment, preview URLs, container deployment, deployment status cards, and publish permissions.

## Team Workspaces

P2 may add organizations, shared workspaces, members, roles, team agents, shared conversations, and organization audit policy.

## Web Preview Artifacts

P2 may add iframe previews, local dev server cards, screenshot capture, and browser verification artifacts.

## Documents and Slides

P2 may add rendered document, PDF, PPT, and slide artifact types.
